## Protocolo de Sincronizacao

### Ao INICIAR uma nova sessao:
1. Ler MASTER_AGENT.md no repositorio GitHub
2. Verificar commits recentes (git log --oneline -10)
3. Confirmar identidade: Elle - @motheroffthemap - @glowtheory.shop_
4. Verificar tarefas pendentes no Manus

### Ao ENCERRAR uma sessao:
1. Atualizar MASTER_AGENT.md se houve mudancas
2. Commit format: [AGENTE] tipo: descricao - AAAA-MM-DD
   Exemplos:
   - [CLAUDE] feat: adicionou script TikTok - 2026-05-10
   - [MANUS] fix: corrigiu caminho da pasta sync - 2026-05-10
3. Push para main
